#### Bower Installation
# Install Bower Dependencies :
#  Bower requires node, npm and git.
#  see here : http://bower.io#install-bower

#Install Bower :
  npm install -g bower

#### How To update Javascripts plugins via Bower :
  cd public_html/scripts/js
  bower update --save

#### How To add a new Javascript plugin via Bower :
  cd public_html/scripts/js
  bower install <package> --save
